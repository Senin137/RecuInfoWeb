Algunas páginas del gobierno que tienen la opción de usar API

**datos.gob.mx** (Plataforma Nacional de Datos Abiertos): Es el repositorio principal, que centraliza información de múltiples dependencias gubernamentales, facilitando la búsqueda de APIs en formatos como CSV o JSON.

**INEGI (Instituto Nacional de Estadística y Geografía)**: Ofrece la API del Banco de Indicadores, la cual permite acceder a datos estadísticos sobre la economía, población y geografía de México de forma gratuita.

**PROFECO (Procuraduría Federal del Consumidor)**: Proporciona una API de "Datos Abiertos" que incluye información sobre precios de productos y servicios.
Transparencia Presupuestaria: Cuenta con una API de Contrataciones Abiertas, útil para consultar licitaciones y procesos de contratación pública.

**Gobierno de la Ciudad de México**: Ofrece APIs locales con datos relevantes de la capital (movilidad, servicios, etc.).

**Diario Oficial de la Federación (DOF)**: Proporciona acceso a sus datos abiertos, incluyendo la publicación de normativas y leyes.

**Banco de México (Banxico)**: Generalmente proporciona acceso a indicadores financieros como tipo de cambio y tasas de interés a través de sus servicios de información.
